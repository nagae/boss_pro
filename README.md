# 分野横断セミナーPro

## 目的
*月に一人の話題提供者* を設けることで、互いの研究成果や各分野の最近の動向~~やおすすめの飲食店~~などに関する知識を分野横断的に共有し、共同研究や萌芽研究課題の申請~~や定期的な飲み会~~に繋げていくための人的ネットワークを構築すること。

当面は、発起人をはじめ、東北大の研究者の持ち回りやその知り合いなどで話題提供していきますが、いずれは、各学会の大御所や有力な若手などもお招きできたらと考えています。
